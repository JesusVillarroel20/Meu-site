Titulo de nuestro documento

Sublime Text trabalha focalizando o que é realmente importante - o código - e deixa os outros tipos de candido para o olho. A interface minimalista que pode ser personalizada com uma dúzia de diferentes esquemas de cores.
Ele também inclui um mini-mapa acessível no lado direito, o que permite que você se mova rapidamente de cima para baixo no seu código. Isso não é visto em outros editores de código e é uma grande vantagem.
